    STAF is test automation framework build on Java using Selenium Web driver. This one of its kind which helps the automation engineer generate scripts easily. It has the flavors of Page Object Model, Data driven and Modular frameworks.

    It has the built in intelligence to load the data automatically from excel to Page actions and identifying objects with Index.
